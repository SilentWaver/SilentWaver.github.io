## 运算符
算术运算符(+ - * / % ++ --),关系运算符(== != > < >= <=),逻辑运算符(&& || !),位运算符(& | ^),赋值运算符

**特殊** : 

| 运算符 | 描述 |
| ---- | ---- |
| sizeof() | 返回数据类型的大小 |
| typeof() | 返回class类型 |
| & | 返回变量地址 |
| * | 变量指针 |
| ?  : | 三目运算 |
| is | 判断某对象是否位某一类型 |
| as | 强制转换,转换失败不会抛出异常 |
